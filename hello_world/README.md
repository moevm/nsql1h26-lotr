# 13. Каталог персонажей вселенной Властелина колец

## Используемые инструменты
Python + Neo4j
Бэкенд будет написан на Django, фронтенд на HTML, CSS, контейнеризация с помощью Docker

## Описание приложения
Веб-приложение предназначено для исследования вселенной Властелина колец через взаимосвязи между персонажами, предметами, локациями и временами. Пользователь будет иметь возможность находить информацию о героях, связанных с ним предметах, локациях, временах, а так же иметь возможность находить связи между персонажами, получить статистику по количеству связей, по признакам
- Графовая база данных — узлы представляют персонажей, предметы, локации, времена. Связи между ними — дружба, владение предметом, появление на локациях и т.д.
- Поиск — возможность искать персонажей по имени, расе (отображение всех связей персонажа с локациями, предметами, временами), предметы, локации, времена
- Аналитика — возможность поиска кратчайшего пути между персонажами
- Агрегация — предоставление статистики: сколько персонажей каждой расы; какие локации, предметы или персонажи имеют больше всего связей; сколько всего персонажей, предметов; статистика персонажей по количеству связей
